# بوت تنبيهات التداول على تليجرام

بيبعتلك رسالة تليجرام في حالتين:
1. قبل خبر اقتصادي مهم (USD أو EUR) بـ 30 دقيقة.
2. لما شرط استراتيجية EMA20/50 + RSI يتحقق على EUR/USD.

يشتغل تلقائياً كل 15 دقيقة عن طريق GitHub Actions - مفيش سيرفر ولا جهاز محتاج يفضل شغال.

---

## الخطوات

### 1. اعمل بوت تليجرام
1. افتح تليجرام وابحث عن `@BotFather`.
2. ابعتله `/newbot` واتبع التعليمات (اختار اسم ويوزرنيم للبوت).
3. هيديك **Token** شكله كده: `123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ` - احتفظ بيه.

### 2. جيب الـ Chat ID بتاعك
1. ابعت أي رسالة للبوت اللي عملته (زي "hi").
2. افتح في المتصفح:
   `https://api.telegram.org/bot<TOKEN>/getUpdates`
   (حط التوكن بتاعك مكان `<TOKEN>`)
3. هتلاقي في الرد رقم `"chat":{"id": 123456789 ...` - الرقم ده هو الـ Chat ID بتاعك.

### 3. ارفع الكود على GitHub
1. اعمل حساب على github.com لو مفيش عندك.
2. اعمل Repository جديد (public - مجاني بالكامل مع GitHub Actions).
3. ارفع كل الملفات اللي في المجلد ده (bot.py, requirements.txt, state.json, .github/).

### 4. ضيف الأسرار (Secrets)
1. في صفحة الـ Repository: Settings → Secrets and variables → Actions.
2. اضغط "New repository secret" وضيف:
   - `TELEGRAM_BOT_TOKEN` = التوكن بتاع البوت
   - `TELEGRAM_CHAT_ID` = الـ Chat ID بتاعك

### 5. فعّل الـ Actions
1. روح تاب "Actions" في الريبو.
2. لو ظهرلك زرار "I understand my workflows, go ahead and enable them" اضغط عليه.
3. اختار "Trading Alerts Bot" من القايمة، واضغط "Run workflow" عشان تجربه فوراً بدل ما تستنى الـ 15 دقيقة.

لو كل حاجة مظبوطة، هتوصلك رسالة على تليجرام (لو فيه خبر قريب أو إشارة استراتيجية متحققة وقت التشغيل).

---

## تعديلات ممكن تحتاجها

- **تغيير التردد**: عدّل السطر `cron: "*/15 * * * *"` في `.github/workflows/scheduler.yml` (الأرقام دي بتحدد كل كام دقيقة يشتغل).
- **تغيير مدة التنبيه قبل الخبر**: غيّر `NEWS_LEAD_MINUTES` في `bot.py`.
- **تغيير حساسية دخول الاستراتيجية**: غيّر `NEAR_EMA_THRESHOLD` في `bot.py`.

---

## تنويه مهم

- الإشارات دي **آلية بناءً على قواعد ثابتة** (EMA/RSI) - مش توصية استثمارية، ولازم تختبرها على حساب ديمو الأول.
- بيانات الأسعار من `yfinance` ممكن تتأخر شوية عن السعر الفعلي عند الوسيط بتاعك - متعتمدش عليها للدخول اللحظي الدقيق، استخدمها كتنبيه أولي وارجع لمنصة التداول بتاعك تأكد.
- مصدر التقويم الاقتصادي غير رسمي وممكن يتأخر أو يفوت حدث نادراً - لو الموضوع حساس بالنسبة لك، راجع تقويم منصتك كمان.
